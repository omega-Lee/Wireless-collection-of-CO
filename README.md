# Wireless-collection-of-CO
# 基于STC单片机的一氧化碳无线采集系统
# 持续更新...
---
## 使用部件：
+ ### 主控：    stc12系列单片机
+ ### 无线模块：NRF24L01
+ ### 一氧化碳传感器：MQ-7
+ ### 显示模块：OLED12864
+ ### 时钟模块：DS1302
+ ### 外部存储：24C02
+ ### 其他部件：蜂鸣器，LED
---
## 编译环境：win10 Keil4  
---

## 文件使用：
>### main.c是运行文件。
>### TX1,TX2是两个从机main文件。
>### RX是主机main文件。
>### 使用方法：为不同芯片下载程序的时候，只需要把把对应程序替换到主main文件就可以。如果需要多从机，只需要把对应从机main文件多复制几份然后修改里面的发送标志就可以了。
